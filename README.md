# Devioz - Sitio Web (Prácticas Pre-Profesionales)


## 📂 Estructura del Proyecto
- `index.html` - Página principal (Frontend)
- `css/styles.css` - Estilos visuales
- `js/main.js` - Animaciones y navegación
- `db.php` - Conexión segura a la base de datos MySQL
- `enviar.php` - Procesamiento lógico de formularios en PHP
- `assets/` - Recursos gráficos y multimedia

## 🛠️ Tecnologías y Herramientas
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Backend:** PHP 8.x
- **Base de Datos:** MySQL (Administrado con MySQL Workbench)
- **Servidor Local:** XAMPP

## ☁️ Infraestructura y Despliegue (Próximas Fases)
- **Contenedores:** Docker (Para empaquetar la aplicación y la BD)
- **Nube:** AWS EC2 (Servidor para el despliegue final)
- **Control de Versiones:** Git & GitHub
